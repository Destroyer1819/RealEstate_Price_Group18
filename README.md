# Real Estate Price Prediction

A machine learning web application that predicts house prices based on property features.

## Project Structure

```
RealEstate_Price_Project/
├── data/
│   └── House_Price_Prediction_Dataset.csv
├── models/                        # Generated by running src scripts
│   ├── lr_model.pkl
│   ├── rf_model.pkl
│   ├── xgb_model.pkl
│   └── feature_columns.pkl
├── notebooks/
│   └── Data_Understanding.ipynb
├── src/
│   ├── data_preparation.py
│   └── model_training.py
├── app.py
├── requirements.txt
└── README.md
```

## Models

| Model | Type | Metric |
|---|---|---|
| Linear Regression | Regression | R², MAE, RMSE |
| Random Forest | Regression | R², MAE, RMSE |
| XGBoost | Regression | R², MAE, RMSE |

**Target variable:** Price (house sale price in USD)

## Setup & Run

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Train models
```bash
cd src
python model_training.py
```

### 3. Run the app
```bash
python app.py
```
Open http://localhost:8050

## Deployment (Render)

1. Push to GitHub
2. Create Web Service on render.com
3. Build Command: `pip install -r requirements.txt`
4. Start Command: `gunicorn app:server`
